# 📰 News Web App

A modern News Web App built using Python, Flask, and NewsAPI.

## 🚀 Features

- Search latest news by topic
- Real-time news updates
- Beautiful modern UI
- Top 10 latest articles
- Deploy-ready project

## 🛠 Technologies Used

- Python
- Flask
- Requests
- NewsAPI
- HTML
- CSS

## 📦 Installation

Install dependencies:

pip install -r requirements.txt

## ▶ Run Project

python app.py

Then open browser:

http://127.0.0.1:5000/

## 🔑 API Key Setup

Open `app.py` and replace:

API_KEY = "YOUR_API_KEY"

with your real NewsAPI key.

Get API key from:
https://newsapi.org/
